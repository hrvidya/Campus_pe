# AI API Integration Project

This project demonstrates integration of multiple AI APIs using Python.

## APIs Used
- Groq
- Google Gemini
- HuggingFace
- Cohere
- Ollama
- OpenAI (optional)

## Features
- Takes user input
- Sends prompt to AI API
- Displays response
- Error handling implemented

## How to Run

1. Install dependencies:
pip install -r requirements.txt

2. Add API keys in .env file

3. Run any file:
python groqass.py
